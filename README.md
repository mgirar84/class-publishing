This my news app that pulls data from https://newsdata.io/ and displays it to the user.
It uses [Ionic](https://ionicframework.com/) and [React](https://react.dev/) 

## Getting Started

Run the development server:

Web:
```bash
npm install
ionic serve
```

Android:
```bash
npm install
npx cap run android
```
